## Klasör yapısı

```
new-assets/
├── goal.md              # Tema hedefi ve eşleme tablosu
├── SOURCES.md           # Bu dosya
├── downloads/           # Ham .zip
├── kenney/              # Çıkarılmış tam paketler
└── curated/             # Oyuna yakın seçilmiş adaylar
    ├── gameplay/
    ├── ui/
    ├── audio/
    └── reference/       # Paket önizleme görselleri
```

## Soft modern tema notu

- **UI Pack** → `PNG/Blue/Default/` ve `PNG/Grey/Default/` — yuvarlak butonlar, slide paneller.
- **Platformer Redux** → `PNG/Backgrounds/blue_*`, `PNG/Tiles/`, `PNG/HUD/` — oyun içi sprite ve can ikonu.
- **Game Icons** → siyah/beyaz minimal ikonlar (2x çözünürlük).
- **Tiny Town** → 16×16 tile; blok dokusu veya referans için.
- **UI Adventure** → ek panel çeşitleri (daha “oyun UI” hissi).

OpenGameArt, Kenney’nin resmi mirror’ı olarak kullanıldı (itch hesabı gerekmeden doğrudan indirme).
